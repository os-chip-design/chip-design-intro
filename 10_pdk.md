## What is a PDK?

 * A collection of files needed to design a chip for a specific technology
 * Provided by the foundry (SkyWater in our case)
 * Needed to produce a GDSII file


## What is in a PDK?

   - LVS: layout versus schematic

 * Standard cell libraries

# Design Rules

 * A set of rules that must be followed when designing the layout of the chip
 * Ensures that the chip can be fabricated correctly
 * Examples:
   - Minimum width of a metal line
   - Minimum spacing between two metal lines
   - Minimum size of a via
 * DRC: design rule check
 * DRC tools check if the layout follows these rules

# DRC Example: Antenna Effect

 * Long metal wires connected to a transistor gate
   - Act as an *antenna* during fabrication
 * Can collect charges during fabrication
 * May damage the transistor's thin gate oxide
 * Foundries provide antenna rules to avoid this problem
 * DRC tool will check for this and flag any violations

# Fixing the Antenna Effect

 * Can be avoided by using shorter wire segments via the router
 * Not possible in LibreLane with the OpenROAD router
 * LibreLane uses antenna diodes instead
 * Inserted next to the cell input pin
   - Provided as a standard cell
